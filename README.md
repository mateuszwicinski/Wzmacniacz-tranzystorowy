# Wzmacniacz Tranzystorowy - Projekt PCB (Eagle)

Projekt zrealizowany w ramach laboratorium na Politechnice Gdańskiej.

**Opis projektu**
Zadanie polegało na przeniesieniu schematu ideowego wzmacniacza tranzystorowego do środowiska Eagle oraz opracowaniu projektu płytki drukowanej. W ramach zadania przetestowano różne metody prowadzenia ścieżek oraz techniki wypełniania wolnych obszarów miedzią.

**Spis plików i ich zawartość**
* **Wicinski.sch** – Schemat ideowy wzmacniacza (plik źródłowy Eagle).
* **Wicinski_fill_miedz.brd** – Projekt płytki drukowanej z zastosowaniem poligonów wypełnienia miedzią.
* **Wicinski_autoroute.brd** – Alternatywna wersja projektu wykonana przy użyciu narzędzia autorouter.
* **Wicinski.lbr** – Biblioteka elementów wykorzystanych w projekcie.
* **Sprawozdanie metody projektowania i technika realizacji.pdf** – Dokumentacja opisująca proces projektowy i realizację zadania.
* **Spis_elementów** – Wygenerowana lista materiałowa (BOM).
* **Spis_polaczen** – Lista połączeń elektrycznych (Netlist).
